# 📘 Miniguia de Estudos com NotebookLM

## 1. Contexto e Objetivos

Este projeto tem como objetivo utilizar o NotebookLM como ferramenta de apoio ao estudo de artigos sobre Inteligência Artificial aplicada à manutenção preditiva industrial.

A proposta é estruturar o aprendizado antes da leitura aprofundada dos artigos, identificando conceitos-chave, tecnologias recorrentes e o contexto da área.

### Objetivos

- Identificar os principais pré-requisitos para leitura dos artigos  
- Organizar os conceitos fundamentais do tema  
- Melhorar a compreensão técnica através de prompts estratégicos  
- Criar um material reutilizável para revisão

## 4. Miniguia de Estudo

### 4.1 Principais Tópicos

**Fundamentos**

- Machine Learning básico  
- Estatística aplicada  
- Séries temporais  
- Conceitos de manutenção preditiva  

**Intermediário**

- Deep Learning  
- NLP (Processamento de Linguagem Natural)  
- Dados de sensores (IoT)  

**Avançado**

- Aplicação de LLMs em contexto industrial  
- Detecção de anomalias  
- Interpretação de modelos  

---

### 4.2 Resumo do Tema

Os artigos analisados mostram o uso crescente de Inteligência Artificial para prever falhas em equipamentos antes que ocorram.

Há forte uso de dados de sensores, modelos de Machine Learning e, mais recentemente, LLMs para apoio na análise e interpretação dos dados.

Os principais desafios incluem:

- Qualidade dos dados  
- Escassez de dados rotulados  
- Complexidade dos sistemas industriais  

---

### 4.3 Glossário

- **Manutenção Preditiva:** previsão de falhas antes que ocorram  
- **LLM:** modelo de linguagem de grande escala  
- **Séries Temporais:** dados coletados ao longo do tempo  
- **Anomalia:** comportamento fora do padrão  
- **Feature Engineering:** criação de variáveis para modelos  

---

### 4.4 Prompts Reutilizáveis

- Quais conceitos preciso entender antes de estudar esse tema?  
- Quais padrões existem entre esses artigos?  
- Explique esse conceito de forma simples e aplicada  
- Quais são as limitações dessas abordagens?  
- Gere perguntas para testar meu entendimento

## 5. Conclusão

O uso do NotebookLM se mostrou eficiente para organizar o aprendizado e identificar lacunas de conhecimento antes da leitura dos artigos.

A qualidade dos resultados depende diretamente da clareza e estrutura dos prompts utilizados.
